WebGL Living Room

Below is a 3D living room scene created using JavaScript's WebGL API as part of a Computer Graphics university submodule

View it on GitHub

The code and instructions for use can be viewed here.

Controls

Please note that these controls only apply to desktop machines

Tilt: ↑ ↓ arrow keys Turn on TV: T
Rotate: ← → arrow keys Toggle lights: L
Zoom in: Z Rotate rug: R
Zoom out: X
